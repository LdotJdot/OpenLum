# 工作机会搜索 Skill（BOSS直聘）

在 BOSS直聘（https://www.zhipin.com/）上按用户要求搜索职位：输入关键词后，在左侧查看职位名称与列表，点击某一职位后在右侧查看详细信息。

**前置条件**：执行本 skill 前必须已加载**浏览器技能**（任一种可进行网页打开、输入、点击、查看页面内容的浏览器自动化技能即可，不限定具体是哪一个）。

---

## 使用场景

- 用户说「找工作」「搜一下 xx 职位」「BOSS直聘搜 Java」「看看北京的前端岗位」
- 用户给出职位关键词、城市、经验等，要求搜索并查看职位列表与详情

## 工作流程

### 1. 用户提供信息

用户会说明搜索条件，例如：
- **关键词**：职位名称（如「Java」「前端开发」「软件测试」）、公司名等
- **城市**：可选，如「北京」「上海」
- **其他**：经验要求、薪资等（若页面支持筛选可后续操作）

若用户未说明关键词，先询问「要搜索什么职位或关键词？」。

### 2. 打开 BOSS直聘并搜索

- 用浏览器技能打开 https://www.zhipin.com/
- 若需登录：可提示用户先登录 BOSS直聘，或按用户提供的凭据完成登录
- 在页面中找到搜索框，填入用户给出的关键词（若有城市选择则一并选择）
- 点击搜索或提交，进入搜索结果页

### 3. 左侧职位列表与右侧详情

BOSS直聘搜索结果页通常为**左右分栏**：

- **左侧**：职位列表，每一项为一条职位（职位名称、公司、薪资等）
- **右侧**：点击左侧某条职位后，右侧显示该职位的详细信息（职责、要求、公司介绍等）

操作顺序：

1. **查看左侧列表**：读取左侧职位名称与简要信息，必要时滚动列表
2. **点击某一职位**：按用户关心的序号或职位名称，点击该条
3. **查看右侧详情**：右侧会展示职位描述、任职要求、公司信息等，将关键信息整理后回复用户
4. 若用户要看更多职位：在左侧再点击其他列表项，重复查看右侧详情并汇总

### 4. 多页与翻页（可选）

若结果有多页，通过「下一页」或分页按钮翻页，继续从左侧列表读取并点击查看右侧详情。

## 页面结构说明（参考）

- **首页**：顶部通常有「职位」「公司」等 Tab，以及搜索框和城市选择。
- **搜索结果页**：左侧为职位列表，右侧为详情面板；点击左侧不同项，右侧内容会切换。
- 若页面改版，以当前页面实际结构为准。

## 注意事项

1. **浏览器技能**：必须确保已加载任意一种浏览器技能，否则无法完成打开页面、输入、点击、查看内容等操作。
2. **登录**：未登录时可能只显示部分职位或无法查看详情，可提示用户登录或由用户提供凭据后完成登录。
3. **操作频率**：操作间隔适当，避免请求过于频繁。
4. **新标签页**：若某次点击在新标签打开，需先切换到对应标签再继续查看。
5. **信息汇总**：从右侧详情提取职位名称、薪资、经验要求、职位描述、公司名等，用自然语言整理后回复用户。
